Instrucciones para la importación del Proyecto

1. Una vez ya está descomprimido el proyecto abrir eclipse(preferiblemente Neon que fue el que se utilizó en el desarrollo del proyecto)
2. Una vez abierto Eclipse, dirigirse a la pertaña file > import
3. En la ventana que se desplegará desplegamos general > Existing Projects into Workspace
4. En Existing Projects into Workspace damos click en browse y navegamos hasta llegar a la carpeta del proyecto que se desea importar
5. Dando un solo click sobre esta (Sin ingresar a esta), seleccionamos aceptar y luego finalizar.
6. A la izqueirda se mostrará el package explore, con los archivos del proyecto para poder navegar por este
